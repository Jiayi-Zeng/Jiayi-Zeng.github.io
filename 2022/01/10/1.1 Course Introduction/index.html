实用机器学习 | 1.1 Course Introduction

January 10, 2022 1166

课程介绍

Industrial ML Applications

Manufacturing Predictive maintenance, quality control
Retail Recommendation, chatbot, demand forecasting
Healthcare Alerts(警报) from real-time patient data, disease identification
Finance Fraud(恶意) detection, application processing(处理申请)
Automobile Breakdown(故障) prediction, self-driving

ML Workflow and challenge

Roles

  • Domain experts(领域专家): have business insights, know what data is important and where to find it, identify the real impact of a ML model
  • Data scientists(数据科学家)：full stack on data mining, model training and deployment
  • ML experts(机器学习专家): customize SOTA ML models
  • SDE(软件开发工程师)：develop/maintain data piplines, model training serving pipeling

Skill improvement

How data scientists spent their time

  • Data loading
  • Data cleansing
  • Data visualization
  • Model selection
  • Model training and scoring
  • Deploying models

Course Topic

Data

  • Collect/ preprocess data
  • Covariate/ concepts/ label shifts
  • Data beyond IID

Train

  • Model validation/ combinations/ tuning
  • Transfer learning 迁移学习
  • Multi-modality 多模态

Deploy

  • Model deployment
  • Distillation

Monitor

  • Fairness
  • Explain ability
